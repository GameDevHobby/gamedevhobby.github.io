## A small site update

I just finished a full refresh of the game jam project pages.

This is not a development update for a new build - it is a content update so the pages better reflect what each project is and how they were made.
I also refreshed the main **Clone Station** project page so it better matches the current state of the game.

## What changed

- Added stronger game descriptions to each jam project page.
- Added clearer section structure (core loop, jam work, reflection, and links).
- Added cover art and screenshot organization where appropriate.
- Split the Projects area so regular projects and game jam projects are easier to browse.
- Expanded the Clone Station page with updated milestones, current status, and a clearer systems overview.

## Clone Station page refresh

The Clone Station project page now includes:

- A clearer core-loop breakdown and feature framing.
- Recent milestone details from the latest devlogs (World 1 completion, enemy additions, input and balance updates).
- Updated status information to reflect the current **Open Beta** stage.
- Direct links for Google Play download and feedback submission.

## Collaboration details now included

I also added collaboration notes for my daughter, **Fae** (now 14), across these projects.

For the earlier projects, she created the title screens and cover art.
By the time we reached **Unwritten.** and **Moth Breaks Everything!!**, she had become deeply involved in design and creative direction, including dialog, world ideas, character work, level design, and learning Blender to model the moth.
